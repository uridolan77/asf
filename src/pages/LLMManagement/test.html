LLM Management

This page provides management functionality for Large Language Model (LLM) components, including LLM Gateway, DSPy, and BiomedLM.

LLM Gateway

Manage LLM providers and configurations, test connections, and execute direct LLM requests. The gateway provides a unified interface for interacting with various LLM providers like OpenAI, Anthropic, and more.

DSPy

Manage DSPy modules for advanced LLM programming. DSPy provides a framework for building complex LLM applications with optimized prompts, fine-tuned for specific tasks like medical RAG, contradiction detection, and more.

BiomedLM

Manage BiomedLM models and adapters for medical-specific LLM capabilities. BiomedLM is a specialized language model for biomedical text, with adapters for specific medical tasks.

Usage

Monitor usage statistics for all LLM components, including token usage, request counts, and latency metrics.
